## 8. ربط Google Sheets

### خطوات الربط:

**الخطوة 1: إنشاء Google Sheets**
1. افتح [Google Sheets](https://sheets.google.com)
2. أنشئ جدول بيانات جديداً باسم: **منصة أسطول الهلال الأحمر - الرياض**
3. أنشئ الشيتات التالية:
   - `Fleet` - بيانات الأسطول
   - `Tracking` - رصد المركبات
   - `Accidents` - الحوادث
   - `Maintenance` - الصيانة الوقائية
   - `OilChange` - تغيير الزيت

**الخطوة 2: إضافة Google Apps Script**
1. في Google Sheets، اذهب إلى: **الإضافات > Apps Script**
2. احذف الكود الموجود
3. انسخ محتوى ملف `google-apps-script.js` والصقه
4. احفظ المشروع
5. انقر على **نشر > نشر كتطبيق ويب**
6. اختر:
   - Execute as: **Me**
   - Who has access: **Anyone**
7. انقر **نشر** وانسخ رابط الـ Web App URL

**الخطوة 3: ربط المنصة بالـ Script**
1. افتح ملف `app.js` في المستودع
2. ابحث عن: `GOOGLE_SCRIPT_URL`
3. استبدل القيمة برابط الـ Web App URL
4. احفظ وارفع التغيير على GitHub

**الخطوة 4: بناء قاعدة البيانات**
بعد نشر الـ Script، اذهب إلى Apps Script وشغّل دالة:
```
setupSheets()
```
هذا سيبني تلقائياً هيكل جميع الشيتات مع الرؤوس الصحيحة.
