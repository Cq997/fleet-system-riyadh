## 🚀 خطوات الإعداد

### الخطوة 1: إعداد Google Sheets

1. افتح [Google Sheets](https://sheets.google.com) وأنشئ ملفاً جديداً
2. من القائمة: **Extensions > Apps Script**
3. احذف الكود الموجود والصق محتوى ملف `google-apps-script.js`
4. احفظ المشروع (Ctrl+S)
5. من القائمة: **Deploy > New deployment**
6. اختر:
   - Type: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
7. انقر **Deploy** وانسخ رابط النشر

### الخطوة 2: ربط المنصة بـ Google Sheets

افتح ملف `data.js` وحدّث السطر التالي:

```javascript
SCRIPT_URL: "https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec"
```

### الخطوة 3: تهيئة الجداول

بعد النشر، افتح الرابط في المتصفح مع إضافة `?action=setup`:

```
https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec?action=setup
```

سيتم إنشاء جميع الجداول تلقائياً مع بيانات الأسطول.

### الخطوة 4: رفع المنصة على GitHub Pages

```bash
# إنشاء مستودع جديد
gh repo create fleet-srca-riyadh --public

# رفع الملفات
cd fleet-srca-riyadh
git init
git add .
git commit -m "🚑 منصة صيانة وتشغيل الأسطول - الإصدار الأول"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/fleet-srca-riyadh.git
git push -u origin main

# تفعيل GitHub Pages
gh api repos/YOUR_USERNAME/fleet-srca-riyadh/pages \
  --method POST \
  -f source.branch=main \
  -f source.path=/
```
